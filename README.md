## Projects 🚀

### <img src="https://cdn-icons-png.flaticon.com/512/919/919851.png" width="16px" /> Cloud Inventory Management
Next.js, Node.js, Prisma, PostgreSQL, AWS (EC2, RDS, Amplify, API Gateway, S3, VPC)
A full-stack inventory dashboard: Next.js + Redux Toolkit frontend on AWS Amplify, Node.js/Prisma backend on EC2, PostgreSQL on RDS in a private subnet, wired together through a custom VPC and API Gateway.
[GitHub](https://github.com/praveen006h)

### <img src="https://cdn-icons-png.flaticon.com/512/2103/2103633.png" width="16px" /> Airline Delay Analysis
Python, XGBoost, scikit-learn
Feature engineering on 5.8M US flight records with cyclic time encoding and stratified 70-10-20 splitting. Benchmarked 5 classifiers — XGBoost led with 88.4% accuracy and 0.94 AUC.
[GitHub](https://github.com/praveen006h)

### <img src="https://cdn-icons-png.flaticon.com/512/2103/2103665.png" width="16px" /> RAG-Chatbot
LangChain, ChromaDB, HuggingFace Embeddings, Groq
A retrieval-augmented chatbot that loads local documents, chunks and embeds them with `BAAI/bge-small-en-v1.5`, stores them in ChromaDB, and answers questions grounded in that data using Groq's `llama-3.1-8b-instant`.
[GitHub](https://github.com/praveen006h/RAG-Chatbot)

### <img src="https://cdn-icons-png.flaticon.com/512/1055/1055666.png" width="16px" /> LangGraph Form Automation
React, Redux, FastAPI, LangGraph, Groq, SQLite
An AI-first CRM module for logging healthcare-professional interactions — either through a structured form or a conversational agent. The LangGraph agent runs 5 tools: log interaction, edit interaction, get HCP interactions, search interactions, and get HCP details.
[GitHub](https://github.com/praveen006h/LangGraph-Form-Automation)
